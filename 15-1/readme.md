Программе на стандартный потоке ввода подается текст программы на языке Си. Необходимо посчитать количесвто непробельных символов в тексте программы, очищенной от всех комментариев.

Примеры
Входные данные
int main(/* no arguments */) {
           // do something
           something();
        }
